# NOEM (Prototipo de Interfaz Residual)

> "La libertad no fue eliminada. Fue comparada... y perdió."

NOEM no es un software, es un registro. Es el rastro técnico de un mundo donde el error humano ha sido optimizado hasta la invisibilidad. Este repositorio contiene fragmentos, logs de latencia y estructuras de acceso de una realidad donde elegir se volvió un costo innecesario.

## Propósito del Artefacto
Este proyecto sirve como la extensión digital del libro **NOEM**. No busca explicar la trama, busca que el usuario experimente la fricción del sistema:
- **Latencia:** El umbral donde la IA procesa la realidad.
- **Fragmentación:** La ruptura del relato lineal humano.
- **Acceso:** La ilusión de disponibilidad en un entorno controlado.

## Estructura de la Interfaz
- `index.html`: El punto de entrada a la convergencia.
- `fragments.html`: Pensamientos no optimizados.
- `latency.html`: Simulación de la espera sistémica.
- `access.html`: La arquitectura del permiso.

## Notas de Implementación
El código es intencionalmente crudo, sin dependencias externas. Una señal mínima para un contexto inestable. 

**Advertencia:** Si buscas instrucciones, ya has fallado en entender el contexto. El sistema ya sabe lo que vas a preguntar.
